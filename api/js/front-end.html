# 前端集成

本节介绍如何充分利用 zkSync 来提供最佳的用户体验。

# 立即投入生产

如果您的前端代码没有部署新的智能合约，那么就不需要对代码库进行任何更改！ 所有现有的 SDK/基础设施都可以开箱即用。

# 启用 zkSync 功能

如果您想部署智能合约或启用高级 zkSync 功能，比如帐户抽象，那么你需要为此使用 zksync-web3 库。 您可以在 此处 阅读基础知识。

如果您想查看一些代码，请查看我们的基本 教程 以获得完整的迷你 dApp。
